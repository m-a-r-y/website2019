Locker Letters

Objective

The goal for this exercise was to gain a further understanding of backend development using Node.js, Express, and APIs.

The Story

The inspiration for this app came from childhood nostalgia when my friends and I used to leave each other anonymous hand written notes. Fast forward to digital modern day where everything leaves a footprint, I wanted a way to bring back some child fun while embracing advancing technology.

Challenges

First Version

One of the challenges for this project was design; I favored a simple and minimalistic design because I wanted the user to focus primarily on their content and message. However, due to the simplicity of the application, there was not much to be left to design other than the form itself.

Pigeon Icon

Working with what I had, I focused mainly on font and typography to convey a playful mood while beckoning the user to write a message. In addition, I wanted a whimsical image for an icon; the little pigeon carrying a letter represents a momentary blast from the past when societies used messenger pigeons to deliver mail.

Final Product

Final Version

Takeaways

This project provided great insight on understanding backend development works; it gives life and functionality to any pertinent application.

To further improve Locker Letters, I would add some animation after the user sends the message by clicking submit. I would have a pigeon fly across the screen and a popup message informing the user that their message has been sent.

Tools Used

• HTML5 • CSS3 • Bootstrap • JavaScript • Node.js • Mailgun API
